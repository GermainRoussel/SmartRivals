# HOLE_TEXT — Texte à Trous

Champ : `holeText: string`. Mettez chaque mot à trouver entre **accolades** `{…}`.
Le joueur reçoit une banque des bons mots (mélangés) à replacer. Pas de `correctAnswer`
(dérivé du `holeText`).
